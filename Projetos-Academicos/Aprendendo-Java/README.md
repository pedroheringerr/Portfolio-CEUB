# Aprendendo Java

Este diretório contém uma coleção de pequenos programas e scripts desenvolvidos com o objetivo de praticar e solidificar meus conhecimentos na linguagem Java.

O foco é cobrir desde a sintaxe básica até conceitos mais avançados de Programação Orientada a Objetos (POO).

## Tópicos Praticados

* Sintaxe básica (variáveis, tipos de dados, operadores).
* Estruturas de controle (condicionais `if/else`, `switch`).
* Loops (`for`, `while`, `do-while`).
* Arrays e Matrizes.
* Conceitos de POO (Classes, Objetos, Métodos).
* Pilares da POO (Encapsulamento, Herança, Polimorfismo).
* Tratamento de Exceções (`try-catch`).
* Coleções (ArrayList, HashMap).

## Ideias de Mini-Projetos para Praticar

Aqui está uma lista de desafios que pretendo implementar para aprimorar minhas habilidades:

-   [X] **Calculadora Simples:** Um programa de console que recebe dois números e uma operação (+, -, *, /) e retorna o resultado.
-   [ ] **Sistema de Gerenciamento de Tarefas (To-Do List):** Um CRUD (Criar, Ler, Atualizar, Deletar) simples no console para gerenciar uma lista de tarefas.
-   [ ] **Pequeno Sistema Bancário:** Simular uma conta bancária com operações de depósito, saque e consulta de saldo, aplicando conceitos de POO.
-   [ ] **Analisador de Texto:** Um script que lê um arquivo `.txt` e conta a frequência de cada palavra.
-   [ ] **Jogo da Velha:** Implementação de um jogo da velha para dois jogadores no terminal.

## Como Compilar e Executar

A maioria dos projetos pode ser compilada e executada diretamente pelo terminal:

```bash
# Para compilar
javac MeuPrograma.java

# Para executar
java MeuPrograma
```
