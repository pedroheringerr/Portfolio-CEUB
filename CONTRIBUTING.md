# Como Colaborar com o PortfolioHUB

Obrigado pelo seu interesse em contribuir! Este projeto segue as práticas de colaboração do GitHub.

## Processo de Contribuição

Para garantir o controle de versão e a segurança do código, todas as alterações devem ser feitas através de Pull Requests.

1.  **Fork:** Faça um "Fork" deste repositório para sua própria conta do GitHub.
2.  **Branch:** Crie uma nova *branch* para sua alteração (ex: `feature/nova-secao` ou `fix/correcao-link`).
3.  **Commit:** Faça suas alterações e realize o *commit* com uma mensagem clara.
4.  **Pull Request (PR):** Abra um Pull Request da sua *branch* para a *branch* `main` deste repositório.
5.  **Revisão:** Sua alteração será revisada antes de ser integrada (merge).

Este processo está protegido pelas "Branch Protection Rules" do repositório.
